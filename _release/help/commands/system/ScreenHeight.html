ScreenHeight%( i%=-1 )

Parameters

  • i - Index of screen. Defaults to the primary screen.

Description

Returns the height of the users screen in pixels.

Example

; ScreenWidth/ScreenHeight
; ------------------------

Print "Your screen is "+ScreenWidth()+" pixels wide."
Print "Your screen is "+ScreenHeight()+" pixels high."
